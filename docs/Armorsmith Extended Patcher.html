Processes ARMO records, modifying them to support Armorsmith Extended

Does not (currently) process Power Armor pieces.

For those records that have already has been modified to support Armorsmith Extended, this should make no changes.

If a records has only had AWKCR keywords added, this should make the armor compatible with Armorsmith Extended.

For records without AWKCR keywords, it reads csv files named .../data/Tools/ArmorsmithExtended_patcher/mod.csv (eg. Fallout4.esm.csv). The intent is that these files can be included with the original mod or a compatibility patch collection.

fieldTypeDescription
armorEditorIDEDIDeditorID of the armor
slotKeywordKYWDAWKCR slot keyword. (eg. _ClothingSlotDevice)
classKeywordKYWDAWKCR class keyword. (eg. _ClothingClassCasual)
slotMaskstringComma separated list of slots numbers this armor uses. If not specified, uses the AWKCR slot keyword to set the slots the armor uses.
addsCarryWeightbooleanSet to Y if the armor should increase carry weight. eg. includes lots of pockets or pouches or includes a pack.
isHighTechbooleanSet to Y if the armor should have thermoptics camoflauge.
